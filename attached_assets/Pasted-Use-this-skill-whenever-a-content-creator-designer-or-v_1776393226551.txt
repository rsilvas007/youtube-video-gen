REGRAS DE OURO — ANTI-COMPRESSÃO
✅ SEMPRE FAZER:

Exportar em 1080p (Full HD) — nunca em 4K para Instagram/TikTok/Facebook
Usar codec H.264 — maior compatibilidade e menor compressão
Formato .MP4 como padrão universal; .MOV para Apple/FCP workflows
30 FPS como taxa segura para todas as plataformas
Bitrate recomendado: 8–15 Mbps para 1080p

❌ NUNCA FAZER:

Exportar em 4K para Instagram — a plataforma recomprime brutalmente
Usar H.265/HEVC como padrão — compatibilidade reduzida
Ultrapassar 4GB por arquivo — limite de upload de algumas plataformas
Ignorar as safe zones — elementos importantes ficam cortados pela UI

ESTRATÉGIA DE MASTER FILE (Para criadores multiplataforma)
Problema: Criar um vídeo diferente para cada plataforma é ineficiente.
Solução: Criar um arquivo mestre e adaptar.
Recomendação de fluxo:
MASTER FILE: 1080 × 1920 (9:16) — vertical
     ↓
     ├── Export 1:1 → cortar laterais para 1080×1080 (Instagram Feed)
     ├── Export 4:5 → cortar para 1080×1350 (Instagram Feed Vertical)
     ├── Export 9:16 → direto (TikTok, Reels, Stories, Shorts)
     └── Export 16:9 → reposicionar elementos (YouTube Padrão)
Por que começar em 9:16?

Vertical é o formato dominante em 2026
Mais fácil cortar vertical → outros formatos
Impossível ampliar horizontal → vertical sem perda de qualidade
